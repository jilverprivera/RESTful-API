### AUTH 
- Sign Up. 
  - `Method: POST` 
  - `Requirements: name | email | password | password2`
  - `Route: ${your_domain}/auth/signup`
- Sign In. 
  - `Method: POST` 
  - `Requirements:  email | password`
  - `Route: ${your_domain}/auth/signin`
